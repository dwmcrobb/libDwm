class Dwm\+::\+Ipv6\+Prefix\+Map$<$ T, Hash $>$}
A wrapper around an unordered\+\_\+map of T keyed by \doxylink{classDwm_1_1Ipv6Prefix}{Ipv6\+Prefix} using Hash as the hash function. 

This isn\textquotesingle{}t the fastest IPv6 prefix container, but... it\textquotesingle{}s fast enough for my uses, and is very little code to maintain since it leverages the C++ standard library facilities.

Members are threadsafe (using a mutex).
