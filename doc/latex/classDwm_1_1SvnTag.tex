Fetches the SVN tag name using a regular expression {\ttfamily tag\+Regex}. 

The default regular expression covers the typical case where a tagged version of software is under a \textquotesingle{}tags\textquotesingle{} directory in the repository. Ditto for fetching a branch name using the regular expression {\ttfamily branch\+Regex} or the trunk using {\ttfamily trunk\+Regex}. When the working copy is on a branch, we will add "{} (branch)"{} to the end of the name to distinguish it from a tag with the same name. When the working copy is the trunk, we will add "{} (revnum)"{} to the end of the name, where revnum is the revision number of the file\textquotesingle{}s last modification. We can only add the revnum if the keyword in the constructor included Header, Id, Revision or a custom keyword substitution whose 3rd word was the revision. For example, "{}\$\+Dwm\+Path\+: dwm/lib\+Dwm/trunk/include/\+Dwm\+Svn\+Tag.\+hh 11210 \$"{}.
