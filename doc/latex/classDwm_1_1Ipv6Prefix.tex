Given that the number of IPv6 addresses I typically deal with is nowhere near 4 billion, a 32-\/bit hash is sufficient if its distribution is effective. 

And it\textquotesingle{}s faster on small 32-\/bit platforms. Hence I\textquotesingle{}m currently using XXH32(), and only on the non-\/zero bytes of the address. Yes, that means I\textquotesingle{}ll have collisions for some prefixes but it\textquotesingle{}s a small (near 0) number in my usage.
