\doxysubsection{Detailed Description}
This class defines an interface for classes that can read their contents from an istream, file descriptor or FILE pointer. 

Note that this class is deprecated as of lib\+Dwm-\/0.\+8.\+0, and is now just a trivial grouping of \doxylink{classDwm_1_1DescriptorReadable}{Descriptor\+Readable}, \doxylink{classDwm_1_1FileReadable}{File\+Readable} and \doxylink{classDwm_1_1StreamReadable}{Stream\+Readable}.
