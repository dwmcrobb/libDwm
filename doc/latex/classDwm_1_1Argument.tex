\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Dwm\+::\+Argument$<$ T $>$}
Encapsulate a single argument. 

Simple class template for a command-\/line argument.

Pre-\/declare the \doxylink{classDwm_1_1Argument}{Argument} class template.

C is the character of the argument, Value\+Type is the type of the argument (bool for arguments that that don\textquotesingle{}t take a value), Req must be true for required arguments.

This can be used directly, thought it typically is only used by \doxylink{classDwm_1_1OptArgs}{Opt\+Args}.
