\doxysubsection{Detailed Description}
This class contains a collection of static functions for reading and writing simple types in network byte order (MSB first) from/to bzip2 files. 

It also contains functions to read and write strings from/to bzip2 files. It also contains function templates to read and write arrays, deques, lists, vectors, maps, multimaps, sets, multisets, unordered\+\_\+maps, unordered\+\_\+multimaps, unordered\+\_\+sets, unordered\+\_\+multisets, tuples and variants from/to bzip2 files. We use our member functions to handle reading and writing simple types in these containers, and function templates to handle reading and writing other class types. In the latter case, the class must implement the \doxylink{classDwm_1_1BZ2Readable}{BZ2\+Readable} and \doxylink{classDwm_1_1BZ2Writable}{BZ2\+Writable} interfaces, since our function templates simply call out to them.
