\doxysubsection{Detailed Description}
This class encapsulates CGI input. 

It is used for storing name/value pairs from CGI input (usually HTML forms) sent via an HTTP POST or GET operation. The constructor does the work of getting the data, hence you should likely only have a single \doxylink{classDwm_1_1Cgi}{Cgi} object in a program. I haven\textquotesingle{}t made this class a true singleton, but may do so in the future.

Under the hood... the implementation uses a trivial flex lexer to get CGI data from the environment (in the case of a GET operation) or stdin (in the case of a POST operation). It also unescapes any escape codes in the input values.
