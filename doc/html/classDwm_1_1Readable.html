Detailed Description

This class defines an interface for classes that can read their contents from an istream, file descriptor or FILE pointer.

Note that this class is deprecated as of libDwm-0.8.0, and is now just a trivial grouping of DescriptorReadable, FileReadable and StreamReadable.
